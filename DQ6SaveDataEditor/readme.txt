# できること
所持金、袋のアイテム(種類、個数)の編集
人間、モンスターの編集(主要パラメータ、名前)

# 初期設定
復号キーがないとセーブデータを読み取れないので、初回使用時に復号キー生成を行う必要がある
端末ごとに復号キーは異なるが、端末で生成した空のセーブデータから生成できる  
空のセーブデータを読み込み、キーの更新ボタンをクリックすれば複合キー(Keys.txtというファイル)が生成される

# 注意点
セーブデータの解析が完全ではないため、値を自由に編集することはできない
変更値の合計が0な状態であれば正常なセーブデータとして保存することができるので、
変更値の合計が0になるよう適当に調整した上で保存する必要がある
